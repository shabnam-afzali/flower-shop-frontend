# melorin-shop-frontend

پروژه فرانت‌اند فروشگاه گل (Flower Shop)
این یک پروژه فرانت‌اند مدرن و جذاب برای یک فروشگاه آنلاین گل و گیاه است که با تمرکز بر زیبایی بصری و تجربه کاربری (UX) طراحی شده است.

ویژگی‌های پروژه
طراحی واکنش‌گرا (Responsive): وب‌سایت به طور کامل برای نمایش در تمامی دستگاه‌ها (موبایل، تبلت و دسکتاپ) بهینه‌سازی شده است.
استایل‌دهی پیشرفته: استفاده از Sass برای مدیریت بهتر کدها، متغیرها و Mixin‌ها جهت ایجاد استایل‌های تمیز و مقیاس‌پذیر.
ساختار استاندارد: پیاده‌سازی با استفاده از تگ‌های معنایی (Semantic HTML) برای بهبود سئو و خوانایی کد.
تکنولوژی‌های استفاده شده
در این پروژه از ابزارهای زیر استفاده شده است:

HTML5: برای ساختاردهی به محتوای صفحات.
Sass (SCSS): برای استایل‌دهی پیشرفته، استفاده از قابلیت‌های Nesting و متغیرها جهت مدیریت تم رنگی و تایپوگرافی.
Flexbox/Grid: برای چیدمان دقیق و منعطف عناصر در صفحه.

## پیش‌نمایش

برای مشاهده برخی از بخش‌های طراحی و رابط کاربری پروژه، می‌توانید اسکرین‌شات‌های قرار داده شده را بررسی کنید.
